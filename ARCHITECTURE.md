## Deployment Scenarios

### Scenario 1: Development (Current)

- Single developer machine
- All services local
- Database in git
- Manual backups

### Scenario 2: Team Shared (Monorepo)

- Monorepo integration
- Shared monitoring stack
- Database in git
- Automated backups via CI/CD

### Scenario 3: Production (Future)

- Dedicated server
- External Prometheus/Grafana
- Database in S3/GCS
- Scheduled backups with retention
